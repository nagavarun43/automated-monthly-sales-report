# Automated Monthly Sales Report

This project automates monthly sales reporting using Python, Pandas, and Excel.

## Overview
The script reads raw sales data from an Excel file, cleans the data, creates monthly summaries, and generates an automated report.

## Files
- `report.py` - main Python script
- `sales.xlsx` - input sales data
- `README.md` - project description

## Features
- Reads sales data from Excel
- Cleans and formats data
- Creates monthly sales summaries
- Generates an Excel report automatically

## Tools Used
- Python
- Pandas
- OpenPyXL
- Excel
