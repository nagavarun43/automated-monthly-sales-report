# Automated Monthly Sales Report

This project is a beginner-friendly Python and Excel automation project.

## Project Overview
The script reads sales data from an Excel file, cleans the data, creates monthly summaries, and generates an automated sales report in Excel.

## Tools Used
- Python
- Pandas
- OpenPyXL
- Excel

## Files in This Project
- `report.py` - main Python script
- `sales.xlsx` - input sales data file
- `monthly_sales_report.xlsx` - generated output report

## Features
- Reads raw sales data
- Cleans and formats the dataset
- Creates monthly sales summaries
- Generates Excel output automatically
- Produces summary insights

## How to Run
1. Install Python
2. Install required libraries:
   `pip install pandas openpyxl`
3. Place `sales.xlsx` in the project folder
4. Run:
   `python report.py`

## Output
The script generates an Excel report with:
- Cleaned Data
- Monthly Summary
- Product Line Summary
- Country Summary
- Insights
